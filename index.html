En este codelab, aprenderás a crear tu primera aplicación con FastAPI y tu primera aplicación con Django para que puedas practicar lo aprendido.

Objetivos

  • Crear tu primera aplicación con FastAPI.
  • Crear tu primera aplicación con Django.

Requisitos previos

  • Conocimientos básicos de Python.
  • Un editor de código instalado en tu máquina.

En este codelab, aprenderás a crear tu primera aplicación web utilizando FastAPI. FastAPI es un framework moderno, rápido (de alto rendimiento) y fácil de usar para construir APIs con Python 3.6+.

Requisitos previos

  • Python 3.6+
  • Pip (gestor de paquetes de Python)

puedes descargar Python desde aquí Python.

Paso 1: Crear un Entorno Virtual

Primero, crea un entorno virtual para aislar las dependencias de tu proyecto.

python -m venv env
source env/bin/activate  # En Windows usa `env\Scripts\activate`

Paso 2: Instalar FastAPI y Uvicorn

Instala FastAPI y Uvicorn (servidor ASGI) usando pip.

pip install fastapi uvicorn

Paso 3: Crear la Estructura del Proyecto

Crea una estructura básica para tu proyecto.

mkdir fastapi_app
cd fastapi_app
touch main.py

Paso 4: Escribir tu Primera Ruta

Abre main.py y define una ruta simple.

# main.py
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello, World!"}

Paso 5: Ejecutar la Aplicación

Ejecuta la aplicación utilizando Uvicorn.

uvicorn main:app --reload

Paso 6: Acceder a tu Aplicación

Abre tu navegador y visita http://127.0.0.1:8000. Deberías ver {"message": "Hello, World!"}.

Paso 7: Agregar una Ruta con Parámetros

Agrega una ruta que acepte un parámetro.

# main.py
@app.get("/items/{item_id}")
def read_item(item_id: int):
    return {"item_id": item_id}

Visita http://127.0.0.1:8000/items/42 para ver el resultado.

Paso 8: Documentación Automática

Accede a la documentación automática generada por FastAPI.

  • Documentación interactiva con Swagger: http://127.0.0.1:8000/docs
  • Documentación alternativa con Redoc: http://127.0.0.1:8000/redoc

Conclusión

¡Felicidades! Has creado tu primera aplicación con FastAPI. Explora más funcionalidades en la documentación oficial de FastAPI aquí FastAPI.

En este codelab, aprenderás a crear tu primera aplicación web utilizando Django, un framework de alto nivel para el desarrollo rápido de aplicaciones web con Python.

Requisitos previos

  • Python 3.6+
  • Pip (gestor de paquetes de Python)

puedes descargar Python desde aquí Python.

Paso 1: Crear un Entorno Virtual

Primero, crea un entorno virtual para aislar las dependencias de tu proyecto.

python -m venv env
source env/bin/activate  # En Windows usa `env\Scripts\activate`

Paso 2: Instalar Django

Instala Django usando pip.

pip install django

Paso 3: Crear un Proyecto Django

Usa el comando django-admin para crear un nuevo proyecto.

django-admin startproject myproject
cd myproject

Paso 4: Crear una Aplicación

Dentro de tu proyecto, crea una nueva aplicación.

python manage.py startapp myapp

Paso 5: Definir una Vista

Abre myapp/views.py y define una vista simple.

# myapp/views.py
from django.http import HttpResponse

def hello_world(request):
    return HttpResponse("Hello, World!")

Paso 6: Configurar las URLs

Configura las URLs para tu vista.

# myproject/urls.py
from django.contrib import admin
from django.urls import path
from myapp.views import hello_world

urlpatterns = [
    path('admin/', admin.site.urls),
    path('hello/', hello_world),
]

Paso 7: Ejecutar el Servidor de Desarrollo

Inicia el servidor de desarrollo de Django.

python manage.py runserver

Paso 8: Acceder a tu Aplicación

Abre tu navegador y visita http://127.0.0.1:8000/hello/. Deberías ver Hello, World!.`

Conclusión

¡Felicidades! Has creado tu primera aplicación con Django. Explora más funcionalidades en la documentación oficial de Django aquí Django.
